Window 对象 ​

由 CreateMainControl 或 CreateChildWindow 返回的对象。

Window:BindWindow(subWindowName, defaultState) ​

仅适用于主控窗口。创建一个按钮，点击后切换指定子窗口的显示/隐藏。

lua
-- 在主窗口中绑定 "战斗功能" 子窗口
Main:BindWindow("战斗功能", false) -- false 表示默认不显示

Window:CreateButton(text, callback) ​

创建一个普通点击按钮。

lua
CombatWin:CreateButton("打印测试", function()
    print("按钮被点击")
end)

Window:CreateModule(text, callback, allowBind) ​

核心组件。创建一个功能模块（类似于 Toggle）。点击开启/关闭。点击右侧 "..." 可展开更多设置（如滑条）。

参数 ​

参数类型说明
textstring模块名称
callbackfunction(bool)回调函数，传入当前开关状态
allowBindboolean(可选) 是否允许按键绑定和长按生成悬浮窗。默认 true

返回值 ​

返回一个 Module 对象，用于添加子组件。

lua
local Aura = CombatWin:CreateModule("杀戮光环", function(state)
    print("光环状态:", state)
end, true)
