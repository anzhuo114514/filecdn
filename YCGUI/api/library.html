Library 全局对象 ​

加载脚本后返回的主要对象，用于创建窗口和管理全局设置。

Library:CreateMainControl(title) ​

创建一个固定在左上角的主控窗口。通常用于导航或总开关。

lua
local Main = Library:CreateMainControl("YC 主菜单")

Library:CreateChildWindow(title) ​

创建一个子窗口（功能页面）。默认隐藏，通常通过主控窗口进行绑定显示。

lua
local CombatWin = Library:CreateChildWindow("战斗功能")

Library:SetupSettings() ​

一键生成内置的UI设置窗口。包含主题切换、缩放、HUD开关等功能。

lua
Library:SetupSettings() -- 建议在脚本末尾调用

Library:Notify(title, status) ​

发送一个右下角的通知弹窗。

参数 ​

参数类型说明
titlestring通知的标题内容
statusbooleantrue显示绿色(开启)，false显示红色(关闭)
lua
Library:Notify("功能已开启", true)
