主题系统 ​

库内置了多种配色方案。你可以通过 Library:SetTheme(name) 或在设置界面中更改。

可用主题 ​

主题名称描述预览
"Default"紫罗兰默认的紫色主题
"Ocean"深海蓝深邃的蓝色主题
"Rose"暗红玫瑰优雅的红色主题
"Sakura"樱花粉温柔的粉色主题
"Mint"薄荷绿清新的绿色主题
"Midnight"午夜橙深沉的橙色主题
"Sunset"日落渐变的日落色调

使用主题 ​

通过代码设置 ​

lua
-- 设置主题
Library:SetTheme("Ocean")  -- 使用深海蓝主题

-- 获取当前主题
local currentTheme = Library:GetTheme()

通过设置界面 ​

使用 Library:SetupSettings() 后，会在设置窗口中提供主题选择下拉菜单，用户可以通过界面直接切换主题。

自定义主题 ​

除了内置主题外，你还可以创建自己的主题：

lua
-- 创建自定义主题
Library:CreateTheme("自定义名称", {
    Primary = Color3.new(1, 0, 0),        -- 主要颜色
    Secondary = Color3.new(0, 1, 0),      -- 次要颜色
    Background = Color3.new(0, 0, 0),     -- 背景颜色
    Text = Color3.new(1, 1, 1)            -- 文字颜色
})

-- 应用自定义主题
Library:SetTheme("自定义名称")

主题与通知 ​

当你更改主题时，所有UI元素（包括通知）都会自动使用新的颜色方案：

lua
-- 切换主题后发送通知会自动匹配新主题
Library:SetTheme("Rose")
Library:Notify("主题已切换到玫瑰红", true)  -- 通知会显示为红色风格

注意事项 ​

  • 主题更改会立即应用到所有窗口和元素
  • 用户设置会被保存，下次运行脚本时会自动恢复上次选择的主题
  • 自定义主题需要在每次脚本运行时重新定义
  • 某些主题在特定设备上可能有更好的视觉效果
