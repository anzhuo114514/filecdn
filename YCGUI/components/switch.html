开关 (Switch) ​

开关是一个简单的二元选择控件，用于启用或禁用功能。在 YC GUI Library 中，可以使用 Module:CreateSwitch 方法创建开关。

创建开关 ​

使用 CreateSwitch 方法创建一个开关：

lua
Module:CreateSwitch(text, callback, default)

参数 ​

参数类型说明
textstring开关旁边显示的文本
callbackfunction(boolean)开关状态改变时执行的回调函数
defaultboolean开关的默认状态（可选，默认为false）

示例 ​

lua
-- 创建一个特效开关
Aura:CreateSwitch("显示特效", function(state)
    print("特效:", state)
    -- 更新特效显示状态
    Settings.showEffects = state
    if state then
        -- 启用特效
        enableEffects()
    else
        -- 禁用特效
        disableEffects()
    end
end, true)

-- 创建一个调试模式开关
Debug:CreateSwitch("调试模式", function(state)
    print("调试模式:", state)
    Settings.debugMode = state
    -- 根据状态显示或隐藏调试信息
    DebugUI.Visible = state
end)

常见用途 ​

  • 功能启用/禁用
  • 特效切换
  • 显示选项控制
  • 调试选项
  • 高级功能控制
